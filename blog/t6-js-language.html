In the JavaScript language, describe all the use cases for each of the following:

  • Parantheses ( )

Parantheses can be used in many different ways such as:

  1. To group expressions (e.g. mathematical operations)
  2. To contain expressions
  3. To contain parameters (e.g. in functions, or variables)

  • Brackets [ ]

Brackets are used for:

  1. Declaring arrays
  2. Accessing arrays
  3. Accessing object properties

  • Braces { }

Braces (or curly brackets) are used for:

  1. Defining the beginning and end of function blocks
  2. Defining initial values in array declarations
  3. Defining the beginning of statements blocks (e.g. for( ) and if( ) structures)

  • Single quotes ' ' and double quotes " "

These are used to create strings. These can be used interchangeably as long as they are used consistently within a string.
