What is grid based design?

Grid based design is when a webpage is comprised of different columns and rows of varying sizes. It will often utilise people's tendency to read a page top to bottom, left to right. Almost every webpage will follow a grid based design, and of those, the majority will consist of 12 columns.

What's all the hype about responsive webpages?

Responsive webpages allow for differing screen sizes. The content in the page will adapt to fit any screen. This is especially important in this day and age as it has become increasingly common to use the internet on phones and tablets, rather than just computer screens.

What is semantic structure?

Semantic structure is a way of organizing elements based on their meaning. For example, a header that uses the class "header". Furthermore, the most important header would use "h1", while the lesser ones would use "h2", "h3", etc. Semantic structures reduce ambiguity for those reading the tags.
